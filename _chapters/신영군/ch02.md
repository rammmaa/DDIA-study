# 3. Code

## 3.1 Imperative vs Declarative Query Languages

Imperative Query language can be explained as the python code, we describe the steps to get the result.
Whereas the Declarative Query language is like the SQL, we just describe the result we want. The processing procedure is hidden.

### Declarative Query Language example

```SQL
SELECT name
FROM employees
WHERE department = 'Engineering'
  AND salary > 80000;
```

### Imperative Query Language example

```python
# Assume 'employees' is a list of dictionaries
result = []
# Step 1: Initialize a loop
for emp in employees:
    # Step 2: Perform a manual check for the first condition
    if emp["dept"] == "Engineering":
        # Step 3: Perform a manual check for the second condition
        if emp["salary"] > 80000:
            # Step 4: Manually update the state of the result list
            result.append(emp["name"])

print(result)
```

# 4. Specific Application Cases & Concept Review

## 4.1 Query Optimizer in Relational Database

In the original article, it didn't deeply described how the query optimizer is constructed, after some research, The query optimizer can be divided into three main parts:

1. Search Space: list all the possible query execution plan
2. Cost Model: estimate the cost of each query execution plan
3. Search Algorithm: find the best query execution plan
   since the database always have to deal with the join request based on million-level data, all the algorithm have to be fast enough to execute, so the query optimizer always being written in C/C++, Go/Rust.

## 4.2 schema on write vs read structure

This 2 structures basically represent the traits of relational model and document model.  
"on write" method is used in relational model, which means the data is validated and structured before it is written to the database. People rule the format of the input data in advance. And all the expected data have to follow the format.  
"on read" method is used in document model, which means the data is written to the database in the format it is given. And the data is validated and structured when it is read.
